代码分析:

  • open( )

open( ) 函数打开文件 file 并返回相应的 file object，在上述例子中即 file_obj。
file 表示将要打开的文件的路径（绝对路径或者当前工作目录的相对路径）
mode 是一个可选字符串，用于指定打开文件的模式。详细参数见下表。

字符 含义
‘r’ 读取（默认）
‘w’ 写入，并先截断文件
‘x’ 排它性创建，如果文件已存在则失败
‘a’ 写入，如果文件存在则在末尾追加
‘b’ 二进制模式
‘t’ 文本模式（默认）
‘+’ 更新磁盘文件（读取并写入）

本例文件的内容是中文，所以指定了编码格式encoding。

  • with 关键字

    简单来说就是，使用 with 关键字可以自动释放文件，该关键字可以让python去确定，我们在不需要访问文件的时候自动关闭文件。正常 open( ) 与 close( ) 总是相对应的，使用了with就不怕忘记写close()语句了。
